# give-lively-api

## Dependencies
* ruby 2.7.0
* rails 6.0.3.2
* postgres ~> 12.0 (https://wiki.postgresql.org/wiki/Homebrew for homebrew users)
* bundler ~> 2.1

## Run Instructions
* start postgres (will be different based on how you installed postgres)
* bundle install
* rake db:migrate
* rails s

## Run specs
* rspec spec
